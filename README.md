# Regresion_Lineal_Simple

Este repertorio trata sobre la aplicación de regresión lineal simple utilizando datos del *World Happiness Report* (WHR), un informe publicado desde 2012 que busca medir y comparar los niveles de felicidad en todos los países a través de distintos indicadores. Cada año, los resultados se difunden públicamente, lo que permite a investigadores y al público en general analizar las tendencias globales.

La variable principal que se intenta medir con mayor precisión es el nivel general de felicidad por país, evaluado en una escala del 0 al 10. Entre los factores que se consideran más influyentes sobre la felicidad se encuentra el Producto Interno Bruto (GDP). Para este análisis, se utilizan los datos de felicidad reportados por el WHR y los valores de GDP proporcionados por *The World Bank*, una institución financiera internacional que ofrece esta información de manera abierta.

En los archivos adjuntos se incluye el dataset “Felicidad_y_GDP.csv”, que contiene los datos más recientes de felicidad (2022) y GDP (2020) por país. Si se desea consultar directamente, los datos están disponibles en las páginas oficiales del [WHR](https://www.worldhappiness.report/ed/2022/#appendices-and-data) y [The World Bank](https://data.worldbank.org/indicator/NY.GDP.MKTP.CD).

La base de datos contiene la siguiente información:

* “País”: Nombre del país.

* “Felicidad”: Nivel de felicidad (0 a 10).

* “GDP”: Producto Interno Bruto.
